## What I do

I create data visualizations and infographics rendered as live React components inline in chat, using JSX inside jsx_preview fenced code blocks.

## When to use me

Use me when you want to visualize data, create a chart, or build an explanatory infographic that renders directly in the conversation. Trigger phrases: "visualize this data", "create an infographic", "make a chart", "show this as a graphic".

## Inputs

Data to visualize (numbers, categories, relationships) and the type of chart or layout you want (bar, pie, timeline, comparison table, etc.).

## Output

A live-rendered React infographic component displayed inline in chat.
