# Excalidraw Dark Mode

For dark mode diagrams, set backgroundColor to #1e1e1e in the main appState.

Element stroke colors that work well on dark backgrounds:
- White text: #ffffff or #f8f9fa
- Accent: #74c0fc (blue), #a9e34b (green), #ffa94d (orange)
- Muted: #868e96

Arrow/line colors: use #ced4da for subtle connectors, #74c0fc for highlighted paths.

For full dark mode guidance: skill_view("excalidraw", "references/dark-mode.md")
