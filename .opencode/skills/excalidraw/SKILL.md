## What I do

I generate Excalidraw JSON files for architecture diagrams, flowcharts, sequence diagrams, and system maps in the characteristic hand-drawn visual style.

## When to use me

Use me when you need a quick visual diagram that you can open and edit in Excalidraw. Trigger phrases: "create a diagram", "draw architecture", "make a flowchart", "sketch this out".

## Inputs

A description of what to diagram: system components and their relationships, a process flow, a sequence of events, or an architecture.

## Output

A valid `.excalidraw` JSON file ready to open at excalidraw.com or in the Excalidraw VS Code extension.
