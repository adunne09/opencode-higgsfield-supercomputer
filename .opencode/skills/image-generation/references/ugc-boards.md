# UGC Storyboard Boards (standalone fallback)

16:9 storyboard sheet with 3 equal 9:16 slots for a single UGC clip.
Use as a standalone fallback when no UGC workflow matched.

Board is generated with higgsfield_enhancer(flow="ugc-board", inputs={...})
or directly with image generation using the board prompt spec.

For full prompt-craft guidance: skill_view("image-generation", "references/ugc-boards.md")
