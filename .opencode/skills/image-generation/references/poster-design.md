# Poster / Banner / Launch Image Design

Use model: gpt_image_2 (best for typography, design, layout)
Typical aspect ratios: 16:9, 9:16, 1:1, 3:4, 4:3

Key rules for posters/banners:
- Text must be exact: quote it literally in the prompt, place it explicitly
- Single-color backgrounds are safest for text legibility
- Hierarchy: headline > sub-copy > details
- Call out font style if important (serif, sans, bold, condensed)
