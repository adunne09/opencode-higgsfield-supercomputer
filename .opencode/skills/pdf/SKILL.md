## What I do

I generate styled, print-ready PDF documents from markdown content, structured reports, or data tables — with control over typography, layout, headers, footers, and visual style.

## When to use me

Use me when you need to export content as a professional PDF. Trigger phrases: "create a PDF", "export to PDF", "generate a report as PDF", "save this as PDF".

## Inputs

Markdown text, structured data, or a content brief. Optional: style preferences (font, colors, layout).

## Output

A downloadable PDF file with the content formatted according to the specified style.
