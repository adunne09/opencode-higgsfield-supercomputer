## What I do

I package the current conversation's workflow into a reusable skill file — capturing the trigger conditions, step-by-step procedure, pitfalls discovered, and verification steps.

## When to use me

Use me after completing a complex task that you want to repeat reliably. Trigger phrases: "save this as a skill", "create a skill", "remember this workflow", "make this reusable".

## Inputs

The completed workflow from the current conversation context. Optional: a skill name and category.

## Output

A properly formatted SKILL.md file written to the appropriate skills directory, ready for future use.
