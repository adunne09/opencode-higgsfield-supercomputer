## What I do

I build complete, content-rich landing pages from a short prompt and publish them live. I generate all visual content (images, video, audio) using AI tools, pick a matching page template, and deploy via React + TanStack Start as a Cloudflare Worker.

## When to use me

Use me when you want a live, deployed website built end-to-end from a single brief. Trigger phrases: "build me a website", "make a landing page", "create a site for", "deploy a landing page".

## Inputs

A short description of the product, service, or brand. Optional: color preferences, target audience, desired sections.

## Output

A live deployed URL with a full landing page including AI-generated hero image, copy, and visual content.
