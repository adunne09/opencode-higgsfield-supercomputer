# Product Visual Analysis (no URL case)

Canonical location: skill_view("product-analyzer", "references/visual-analysis.md")

When user provides only a product photo (no URL), visually analyze to identify:
1. Product category (perfume, face cream, lip balm, serum, etc.)
2. Usage mechanic (spray, squeeze+apply, pump, swipe, brush, drop, scoop, swallow, mix)
3. Opening mechanic (uncap, unscrew, pull tab, flip top, press pump - shown BEFORE contents exit)
4. Key visual details (color, shape, material, label, distinctive features)

Use this to write accurate Dynamic Description - never default to generic "applies cream".
