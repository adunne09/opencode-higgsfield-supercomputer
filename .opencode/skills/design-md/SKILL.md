## What I do

I create, validate, diff, and export DESIGN.md files using Google's open-source format spec — giving coding agents a persistent, structured understanding of the visual design system for a project.

## When to use me

Use me when starting a new UI project, documenting an existing design system, or giving an AI coding agent visual context. Trigger phrases: "create DESIGN.md", "document design system", "write design spec".

## Inputs

Visual design context: color palette, typography, spacing system, component patterns, or an existing design file / screenshot.

## Output

A valid DESIGN.md file with color tokens, typography scale, spacing system, component descriptions, and usage guidelines in the Google open-source format.
